  UART1 — SIM800L GSM Module
  ──────────────────────────
  ESP32 GPIO27 (RX1) ──── SIM800L TX
  ESP32 GPIO26 (TX1) ──── SIM800L RX

  ⚠️  IMPORTANT: SIM800L Power Requirements
  ──────────────────────────────────────────
  • SIM800L requires 3.4V – 4.4V (NOT 3.3V!)
  • Peak current draw: 2A during transmission
  • MUST use LiPo battery directly (4.2V – 3.7V)
  • Add 1000μF bulk capacitor + 100nF decoupling at VCC
  • DO NOT power from ESP32 3.3V regulator!

                    ┌──────────────┐
                    │   SIM800L    │
                    │   GSM        │            ┌─────────┐
                    │              │            │ LiPo    │
                    │ TX ─── GPIO27 (RX1)      │ Battery │
                    │ RX ─── GPIO26 (TX1)      │ 3.7-4.2V│
                    │ VCC ── LiPo+ ◄───────────┤ +       │
                    │ GND ── GND ◄─────────────┤ -       │
                    │       │                   └─────────┘
                    │  ┌────┴────┐
                    │  │1000μF   │  (Bulk capacitor for
                    │  │ 6.3V    │   2A peak current)

                    │       │
                    │      GND

POWER SYSTEM
━━━━━━━━━━━━

                   ┌───────────────────────────────────────────────┐
                   │                                               │
  USB-C ──►  ┌─────────┐   ┌──────────┐   ┌──────────────────┐   │
  5V Input   │ TP4056  │   │ LiPo     │   │ AMS1117-3.3V     │   │
             │ Charger │──►│ Battery  │──►│ LDO Regulator    │   │
             │         │   │ 2000mAh  │   │                  │   │
             └─────────┘   │ 3.7-4.2V │   │ OUT: 3.3V ──────┼───┤
                           │          │   └──────────────────┘   │
                           │          │                          │
                           │ Direct ──┼──► SIM800L VCC           │
                           │ (4.2V)   │    (Needs 2A peak!)     │
                           │          │                          │
                           └──────────┘                          │
                                                                 │
                           3.3V Rail ◄───────────────────────────┘
                             │
                             ├── ESP32 VCC
                             ├── ADXL345 VCC
                             ├── MAX30102 VCC
                             ├── DRV2605L VCC
                             ├── DS18B20 VCC
                             ├── NEO-6M GPS VCC
                             ├── Ra-02 LoRa VCC
                             ├── PAM8302 VCC
                             └── MicroSD VCC

  ⚠️  CRITICAL NOTE:
  SIM800L MUST be powered from LiPo directly (not 3.3V rail).
  During GSM transmission, it draws 2A peak current which will
  cause brownout on any LDO regulator. Use a 1000μF electrolytic
  capacitor + 100nF ceramic capacitor at SIM800L VCC pin.
